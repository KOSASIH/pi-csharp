# pi-csharp

Pi Network C# client library for server side applications.

- Create new payment
- Get payment
- Approve payment
- Complete payment
- Cancel payment
- Get incomplete server payments
- Make transaction for withdraw request.

Require

- API Key from Pi Network
- App Seed Wallet when make A2U transactions.
